<div class="code-title">
    upper()
</div>

> NOTE: `upper()` 함수를 이용하여 문자열을 대문자로 변환할 수 있다.

Example:
```python
x = 'asdAAAd'.upper()
print(x)
```
Out :
```
ASDAAAD
```
<br>

<div class="code-title">
    lower()
</div>

> NOTE: `lower()` 함수를 이용하여 문자열을 소문자로 변환할 수 있다.

Example:
```python
x = 'asdAAAd'.lower()
print(x)
```
Out :
```
asdaaad
```
<br>
